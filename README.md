# 🌐 Landing Page

A modern and responsive landing page built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates the fundamentals of front-end web development with a clean and user-friendly design.

## 🚀 Features

- Responsive layout
- Clean and modern UI
- Navigation bar
- Hero section
- Call-to-action button
- Easy to customize
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Landing-Page/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Open the project folder.

3. Double-click `index.html` or open it with **Live Server** in VS Code.
